# Houzeo Real Estate Assignment - React + TypeScript

A responsive real estate search page with an interactive US map and dynamic property listings. Built with **React**, **TypeScript**, **Tailwind CSS**, and **Mapbox GL JS**.

## Features

- Interactive US map centered on Albuquerque with dynamic property markers.
- Property list on the right with images, details, and hover effects.
- Responsive design: desktop and mobile layouts.
- Hover animations on property cards and favorite icons.
- Lazy loading for images to improve performance.
- Filter bar with smooth hover transitions.
- Sort-by dropdown (UI only, no functionality).
- Bonus: Map supports zoom and pan with dynamically loaded property markers.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Sh0aibShaikh/houzeo-assignment.git
cd houzeo-assignment
````

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm start
```

* App runs at [http://localhost:3000](http://localhost:3000)
